# 🛒 E-Commerce Sales Dashboard

A cloud-based data pipeline and interactive dashboard built on AWS.

## 🔗 Live Dashboard
[Click here to view the dashboard](https://maggi15.github.io/ecommerce-aws-dashboard/)

## 🏗️ Architecture
Raw CSV → Amazon S3 → AWS Glue → Amazon Athena → Python + Plotly

## ☁️ AWS Services Used
- **Amazon S3** — Cloud storage for raw data
- **AWS Glue** — Automated schema discovery and ETL
- **Amazon Athena** — Serverless SQL queries on S3
- **Python + Plotly** — Interactive dashboard

## 📊 Key Metrics
- Revenue by product category
- Monthly revenue trend (2024)
- Revenue by region
- Order status breakdown

## 🛠️ Tools & Technologies
Python | Pandas | Boto3 | Plotly | AWS | SQL | GitHub Pages
